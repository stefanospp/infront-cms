# Language Connections — Claude Code Reference

Private English lessons institute in Paralimni, Cyprus. Custom design based on BrightMind Framer template.

## Key files
- `site.config.ts` — site identity, nav, footer, SEO, theme
- `src/styles/global.css` — brand tokens: blue primary, purple secondary, green accent; DM Sans + Inter
- `src/components/Hero.astro` — custom Hero override (BrightMind split hero with floating badges)
- `src/components/CTA.astro` — custom CTA override (avatar stack + CTA banner)
- `src/components/ProgramCards.astro` — English program cards (Young Learners, Teens, Adults, Exam Prep)
- `src/components/StatsAbout.astro` — stats section with image + floating badge

## Tier: Static
- No CMS — all content is in .astro page files
- Contact form via API route (`/api/contact`) with Resend email

## Design
- BrightMind-inspired modern education aesthetic
- Blue (#2563eb), green (#10b981), purple (#9333ea)
- DM Sans headings, Inter body
- Rounded borders, sticky nav, multi-column footer
